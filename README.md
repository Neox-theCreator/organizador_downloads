# 📁 Organizador de Downloads

Script em Python que organiza automaticamente sua pasta de Downloads (ou qualquer outra) movendo arquivos para pastas baseadas na extensão.

## ✨ Funcionalidades

- 🚀 Move arquivos para pastas como Imagens, Documentos, Videos, etc
- 🔍 Modo `--dry-run` para simular antes de executar
- ↩️ Comando `--undo` para desfazer a última organização
- 📝 Configuração personalizável via JSON
- 📊 Log de todas as operações

```bash
git clone https://github.com/seuusuario/organizador-downloads.git
